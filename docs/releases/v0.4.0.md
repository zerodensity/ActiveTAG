## Product Profiles

- Added separate `CAM`, `Talent Track`, and `April Tags` tabs.
- Preserved the six CAM profiles for Label Groups `0-5`.
- Added 15 Talent Track profiles for Label Groups `6-20`.
- April Tags currently provides a `Custom` profile and is ready for the Far
  and Near profile values.

## Automatic Detection

- Connected CAM configurations automatically open the CAM tab.
- Connected Talent Track configurations automatically open the Talent Track
  tab after validating Uplink ID, LED 0, and disabled LED 1-7 values.

## LED IDs

- LED Active ID editors now display hexadecimal values.
- The corresponding decimal value is shown directly below each LED ID.
- `0xFFFFFFFF` is identified as `Disabled`.
- Ready-made CAM and Talent Track profiles keep all LED ID fields locked.
- Selecting `Custom` unlocks the LED ID fields for manual configuration.

- Passed all 15 Talent Track profile detection and disabled LED tests.
- Built successfully with MSVC C++20 and the static `/MT` runtime.
