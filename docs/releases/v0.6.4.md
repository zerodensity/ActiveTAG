## Robust Cable Pull Detection

- Added a quiet ActiveTAG heartbeat while a device is connected.
- The app no longer trusts the Windows COM-port list alone, because unplugged
  USB serial devices can remain visible in `SERIALCOMM` temporarily.
- If the connected tag stops responding to the heartbeat, the app closes the
  serial handle, clears the stale device UI, and returns to automatic probing.
