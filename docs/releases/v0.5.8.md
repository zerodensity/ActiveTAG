## Talent Tracker LED Mapping

- Talent Tracker profiles now use physical LED 4 as the only active LED output.
- Talent Tracker LED 0, LED 1, LED 2, LED 3, LED 5, LED 6, and LED 7 are written
  as disabled.
- Disabled LED outputs are now written as `0xFFFFFFFF` (`4294967295`).
- The parser still accepts older `0x7FFFFFFF` disabled dumps for backward
  compatibility.

## Documentation

- Updated the README to describe the ActiveTAG disabled LED value and the new
  Talent Tracker LED 4 behavior.

## Verification

- Passed native parser tests, including the Label Group 6 example where LED 4
  uses `0x804`.
- Built successfully with MSVC C++20, Dear ImGui, and DirectX 11.
