## Lens Profiling Profiles

- Activated the Lens Profiling product family with `Profile TAG 1` and
  `Profile TAG 2` ready-made profiles.
- Lens Profiling profiles now write LEDs 0-3 as active IDs and LEDs 4-7 as
  disabled with `0xFFFFFFFF` (`4294967295`).
- Ready-made Lens Profiling profiles lock LED ID fields while keeping Uplink ID,
  RF Channel, Signal Intensity, LED Brightness, and On While Charging editable.

## Detection

- Connected tags matching Lens Profiling LED patterns are automatically opened
  on the Lens Profiling product view with the matching profile selected.
- Custom profile conflict checks now include Lens Profiling LED IDs.

## Verification

- Added native parser coverage for both Lens Profiling profiles and legacy
  disabled LED dump compatibility.
