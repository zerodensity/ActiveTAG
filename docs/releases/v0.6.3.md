## Connection Removal Detection

- Added a connected COM-port watchdog that checks whether the active serial
  port still exists while the application is running.
- When the USB cable is unplugged or the virtual COM port disappears, the app
  now disconnects the ActiveTAG, clears the stale device state, and returns to
  automatic probing.
- Serial read failures and command timeouts now close the serial handle so the
  UI cannot remain stuck in a connected state after a cable pull.
