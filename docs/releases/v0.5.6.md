## Interface Updates

- Renamed the `CAM` product button to `Camera Tracker`.
- Renamed the `Talent Track` product button to `Talent Tracker`.
- Kept the existing segmented button sizing unchanged.

## Safety

- Custom profiles now block writes when the Uplink ID matches a known Label
  Group value from the built-in profile list.
- Custom profiles now block writes when any visible LED Active ID reuses an
  active LED ID reserved by the known Camera Tracker or Talent Tracker profiles.
- Blocked writes show the in-app Dear ImGui error modal: `This ID is in use!`.

## Verification

- Passed native parser tests.
- Built successfully with MSVC C++20, Dear ImGui, and DirectX 11.
