## Talent Tracker LED Mapping

- Moved Talent Tracker ready-made profiles from physical LED 4 to physical LED 3.
- Talent Tracker now writes LEDs 0, 1, 2, 4, 5, 6, and 7 as disabled with
  `0xFFFFFFFF` (`4294967295`).
- The Talent Tracker editor now shows only `LED 3 Active ID`.
