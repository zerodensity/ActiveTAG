## New

- Added persistent serial communication logging next to the executable.
- Added local timestamps with millisecond precision to every physical log line.
- Added complete `TX` command and `RX` device-response recording.
- The log file is kept open while the application runs and closed cleanly on exit.
- Existing log contents are preserved; every application session appends new entries.

## Interface

- Reduced the application window and control layout by an additional 10%.
- Reduced the standard and title font sizes to match the more compact layout.

## Log File

The application creates the following file in the same folder as the EXE:

```text
ActiveTAG-Configurator.log
```

The application must be placed in a folder where the current Windows user has
write permission.
