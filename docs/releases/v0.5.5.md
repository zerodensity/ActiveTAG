## Serial Reliability

- Added one automatic COM-port reopen and retry when Windows rejects a serial
  write with recoverable USB serial errors such as Windows error `22`.
- If the retry also fails, the stale serial handle is closed so the next action
  starts from a clean reconnect.

## Talent Track Disabled LEDs

- Talent Track profiles now write the firmware-safe disabled LED value
  `0x7FFFFFFF`.
- Profile detection accepts both `0x7FFFFFFF` and the older `0xFFFFFFFF` as
  disabled LED values, so connected tags that echo either form can still match
  their ready-made profile.

## Verification

- Passed native parser tests, including both disabled LED sentinel variants.
- Built successfully with MSVC C++20, Dear ImGui, and DirectX 11.
