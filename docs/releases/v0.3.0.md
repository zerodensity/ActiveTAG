## New

- Added `CAM1` through `CAM6` names to the Label Group profile selector.
- Added coordinated native Light and Dark themes, including the Windows title bar.

## Profile Protection

- Selecting a camera profile applies the verified OptiTrack Label Group LED IDs.
- LED Active ID fields `[D0]` through `[D7]` are locked while a camera profile is selected.
- Selecting `Custom` unlocks the LED Active ID fields for manual configuration.
- Uplink ID, RF Channel, LED Brightness, and On While Charging remain editable for every profile.

## Interface

- Unified window and control background painting to remove the dark gray strip and mismatched gaps.
- Preserved the compact native Win32 layout and standalone static executable.

## Verification

- Built successfully with MSVC C++20 and the static `/MT` runtime.
- Passed all native parser tests using the six captured firmware `2.3.4` Label Group configurations.
