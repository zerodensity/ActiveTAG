## Profile Selection

- Profile detection now applies the product tab, matching profile selection,
  and LED field lock state as one operation.
- A connected CAM configuration that exactly matches a known LED ID profile
  displays that CAM profile in the dropdown.
- A connected Talent Track configuration that exactly matches its Uplink and
  LED ID profile displays that Talent Track profile in the dropdown.
- `Custom` is selected only when no known profile matches or when the user
  explicitly chooses it.
